# What I Love – Paper Crafts & Home Decor

This is my front-end project for the Nexus Bootcamp.  
I built a small website about something I really enjoy: **paper crafts and home decoration**.

## What's inside

- 4 pages: Home, About, Gallery, Contact  
- One CSS file for each page (keeps things organised)  
- No JavaScript – only HTML and CSS  

## What you can see

- Why I love paper crafts  
- My favourite materials and the steps I follow to make paper flowers    
- A gallery of my projects (using CSS Grid)  
- A contact form with different input types  

## Features I practiced

- Flexbox and Grid layouts  
- Responsive design (works on mobile, tablet and desktop)  
- Hover effects, transitions, and a small animation  
- A background image with an overlay  
- Internal and external links  
- Pseudo-classes and a pseudo-element
